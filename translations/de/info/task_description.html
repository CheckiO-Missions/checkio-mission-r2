Der zweite Teil der Serie über das japanische Spiel "riichi mahjong". Es ist ein wenig komplizierter als das vorherige Spiel.

Ihre Aufgabe ist es, zu überprüfen, ob das Blatt ein Gewinnblatt ist. Wenn Sie in der vorigen Mission ein fertiges Blatt erhalten haben, dann gibt es in dieser Mission nur eine Reihe von Spielsteinen in den Eingabedaten, die möglicherweise nicht siegreich sind.

Ich möchte Sie an die Bedingungen für den Sieg erinnern.

  • Das Blatt muss genau 14 Spielsteine enthalten.
  • Die Hand muss vier Sets (pon oder chi) und ein Paar enthalten.

In dieser Mission wird das zweite Argument der Funktion hinzugefügt - eine Liste von sogenannten offenen Sets. Ein solches Set entsteht im Spiel durch das Wegnehmen eines Plättchens, das der Gegner als unnötig abgeworfen hat. Die Menge, die durch dieses Plättchen ergänzt wurde, wird beiseite gelegt und verändert sich nicht mehr. Beachten Sie, dass Paare nicht offen sein können.

Jedes der offenen Sets muss überprüft werden, ob es gültig ist oder nicht. Zum Beispiel ist 's456' ein gültiges Set und 'm124' oder 'p11' ist ungültig.

Eine weitere Anmerkung: Es gibt Ausnahmen von der zweiten Siegbedingung. Es gibt zwei Arten von Blättern, die gewinnen können und gleichzeitig nicht vier Sätze und ein Paar enthalten (in einem echten Spiel sind diese Blätter recht selten). Hier sind sie:

  • Chitoisu. Hand, die sieben Paare enthält. Ein Beispiel:
    • ['p1', 'p1', 'p5', 'p5', 'm2', 'm2', 'm6', 'm6', 's3', 's3', 's9', 's9', 'ww', 'ww']

    Beachten Sie, dass die Hand nicht zwei identische Paare enthalten darf.

  • Kokushi-muso. Hand, die eine und neun aus jeder der Farben, jedem der Drachen, jedem der Winde und ein Paar zu einer dieser Figuren enthält. Zum Beispiel:
    • ['m1', 'm9', 'p1', 'p1', 'p9', 's1', 's9', 'wn', 'we', 'ws', 'ww', 'dg', 'dr', 'dw']

Eingabedaten: Zwei Argumente vom Typ Liste: Das erste sind Strings, die Kacheln kodieren, das zweite sind Strings, die offene Mengen kodieren (können leer sein).

Ausgabedaten: Eine Variable vom Typ bool: die Antwort auf die Frage "Ist die Hand bereit?".

Beispiel: 

    riichi_mahjong_win_hand(['m1', 'm2', 'm3', 'm5', 'm5', 'p2', 'p3', 'p4', 'p7', 'p8', 'p9'], ['s456']) == True
